🧹 Data Cleaning Project using Python

📌 Overview

This project demonstrates a complete **data cleaning and preprocessing pipeline** using Python. The objective is to transform raw restaurant advertisement film client request data into a clean, consistent, and analysis-ready dataset suitable for exploratory data analysis (EDA), visualization, and business intelligence.

The project follows industry-standard data preprocessing techniques commonly used in data analytics and data science workflows.

✨ Features

* Import and inspect raw CSV data
* Detect and remove duplicate records
* Handle missing values in numeric and categorical columns
* Remove extra whitespace from text fields
* Standardize text formatting
* Correct common spelling inconsistencies
* Convert date and numeric data types
* Detect budget outliers using the IQR (Interquartile Range) method
* Create new analytical features:

  * Month
  * Quarter
  * Budget Category
  * Premium Project Flag
  * Project Complexity
* Generate summary statistics
* Calculate business metrics such as total revenue and average budget
* Export the cleaned dataset for further analysis

🛠️ Technologies Used

* Python
* Pandas
* NumPy

📂 Project Workflow

1. Load the dataset.
2. Explore the dataset structure.
3. Check for missing values and duplicates.
4. Clean and standardize the data.
5. Convert appropriate data types.
6. Handle missing values.
7. Detect outliers.
8. Perform feature engineering.
9. Generate summary statistics.
10. Export the cleaned dataset.

📊 Output

The project produces:

* A fully cleaned dataset
* Summary statistics
* Revenue summary
* Business-ready data for EDA and visualization
